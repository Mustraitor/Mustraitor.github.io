C语言程序设计笔记

C语言笔记

1. 基本语法

基本数据类型

  • int 4B 默认有符号signed

  • n 位补码表示范围：$[-2^{n-1}, 2^{n-1}-1]$

  • float

  • double

  • long double

符号 代表类型 对应含义
%d Integer 有符号十进制整数。
%s String 字符串。
%c Character 单个字符。
%f Float 浮点数。
%u Unsigned 无符号十进制整数。
%p Pointer 指针。

全局变量初始值为0

  • 定义 (Definition)：编译器为变量分配内存。例如 int a = 10;，这会在内存的数据段（Data Segment）占 4 个字节。

  • **声明 (Declaration)**：告诉编译器变量的类型和名字，但不分配内存。

extern 修饰符的作用就是声明

register修饰符的作用是存储变量到寄存器中

static修饰符

  • 修饰局部变量
  • 修饰全局变量，修饰函数

作用：限制作用域（只在当前文件可见）

用法 作用
static 局部变量 生命周期变长（函数调用后不销毁）
static 全局变量 限制作用域（仅当前文件）
static 函数 限制作用域（仅当前文件）

#define 宏定义

作用：在编译之前，由预处理器进行文本替换的工具

const 常量

十进制数 ：52

八进制数：052

十六进制数：0x52

scanf 输入函数

  • & 取地址运算符

初始化 = 在变量创建（声明）时就给它一个初始值

  • static 修饰 初始值为0
  • 全局变量 初始值为0
  • 局部变量 初始值为随机数( static 修饰为0)

词法分析器：把代码拆成“有意义的最小单位（Token）

位运算

  • & 与运算（有0出0，全1出1）

  • | 或运算（有1出1，全0出0）

  • ~ 非运算

  • << 左移运算符

左移相当于 left * 2^right, 移动后补0

  • 右移运算符 >>

右移相当于 left / 2^right, 移动后补0

  • ^ 异或运算XOR（Exclusive OR）
  • printf()函数有返回值，成功返回打印的字符数

指针（pointer）

  • 表示地址
  • *的作用：声明，解引用

  • 指针未指向时，不允许赋值
  • 返回指针的函数，不能返回局部变量的地址

指针移动

  • 指针移动到下一个元素，并不是直接 +1 还要看数据类型（int 占4字节）
